# Optimizador_De_Componentes_RLC
Este proyecto consiste en un programa Java que dado un inventario de resistencias, capacitores e inductores y un valor de uno de estos deseado, halla la combinación en serie o paralelo o mixta que minimiza la cantidad de componentes necesarios para obtener el valor deseado y muestra el circuito.

## Descripción del problema
En ingeniería electrónica y mecatrónica, frecuentemente se necesita un valor específico de resistencia, inductor o capacitor que no es comercial. Este programa permite gestionar un inventario de componentes pasivos y, a partir de un valor deseado y una tolerancia, encuentra la combinación óptima en serie/paralelo (circuito mixto) que utiliza la menor cantidad de componentes disponibles, minimizando el error.
